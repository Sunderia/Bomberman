# Bomberman
A bomberman game in Minestom

## Requirements:
  - Java 21

## Usage
Download the `bomberman-server.jar` file from the [latest release](https://github.com/Sunderia/Bomberman/releases/latest)
Then run `java -jar bomberman-server.jar`
